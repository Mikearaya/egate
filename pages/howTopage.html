In Order to create an event on the system you should follow the following steps.

If you have alerady registered and has an active account skip 1 and 2

  • you create account by Signing Up read tutorial if you dont know how to
  • Log Into the Account you created read tutorial if you dont know how to
  • Select Create Event
  • Enter details of the event
  • select Save event if you want to make your event live simulatainiously or save draft if you want to make the event available on late use
  • Wait for the Confirmation & You are done if you provided everything correctly

To get newsletter every week about the events happening around you you can create subscription to the contents of your interest

  • Select subscribe button found on the footer of the page
  • Provide your e-mail address and select your request type i.e. New, Update or Remove subscription
  • In the Case of new Select Event categories you want to get newsletter about or if you are updating your subscription change your previous setting
  • If You delete the content will be removed and you will not get anymore newslettrs from Egate
  • Save and exit, You Are Done
